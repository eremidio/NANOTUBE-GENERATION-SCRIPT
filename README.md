# NANOTUBE-GENERATION-SCRIPT

THE PRESENT REPOSITORY CONTAINS SOME PYTHON SCRIPTS TO GENERATE SUPERCELLS EXTENDED ALONG TWO DIRECTIONS DETERMINED BY LATTICE VECTORS AND THEN TO GENERATE A NANOTUBE OR A NANOSCROLL FROM THIS SUPERCELL. THE SCRIPTS WERE MADE BASED ON THE FORMULAS CONTAINED IN THIS ARTICLE: https://doi.org/10.1016/j.jtice.2016.08.030 


THE MAIN PURPOSE IS TO PROVIDE A WORKING SCRIPT WITH MINIMUM REQUIREMENTS. ONLY THE ASE PYTHON LIBRARY IS REQUIRED
IN ORDER TO USE THESE SCRIPTS (SEE THE DOCUMENTATION ON HOW TO INSTALL IT FOR A GIVEN OPERATING SYSTEM IN: https://ase-lib.org/).


THE SCRIPTS ARE MADE TO HANDLE POSCAR FILES USED BY VASP SOFTWARE. SOME TOOLS LIKE VESTA CAN BE USED TO GENERATE/CONVERT
VASP POSCAR FILES FROM/TO OTHER FILE EXTENSIONS (A SMALL SCRIPT TO GENERATE .cif FILES IS ALSO AVAILABLE). ARMCHAIR/CHIRAL AND 
ZIGZAG NANOTUBES ARE TREATED SEPARATELY. NANOSCROLL GENERATION ARE AVAILABLE ONLY FOR THE ARMCHAIR/CHIRAL SCRIPTS. AN ADDITIONAL BASH SCRIPT IS PROVIDED FOR RENAME THE RESULTING POSCAR FILE AND CLEAN THE DIRECTORY FOR MULTIPLE RUNS (REMOVING SOME INTERMEDIATE GENERATED FILES).
